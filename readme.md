# 智能药箱

> **完整项目说明请阅读 [`README.md`](README.md)**（硬件引脚、STM32 工程、协议、取药流程等）。

本文件保留 **ESP8266 模块**的快速参考；与 STM32 联调时请以 `README.md` 中的最新协议为准。

---

## ESP8266 核心功能

- **精准授时**：NTP 客户端，腾讯时间服务器
- **移动端 Web UI**：浏览器设置三路闹钟
- **掉电记忆**：闹钟配置写入 Flash
- **串口协议**：9600 波特率，英文逗号分隔字符串

## 环境依赖（Arduino IDE）

1. 开发板支持包：`esp8266` by ESP8266 Community（3.0.0+）
2. 库：`NTPClient` by Fabrice Weinberg

## 快速上手

1. 修改 2.4GHz Wi-Fi 账号密码并烧录 ESP8266
2. 串口监视器 **9600** 查看 IP
3. 浏览器访问 IP 进入「闹钟控制中心」

## 串口协议（ESP8266 → STM32）

**心跳（须含 IP，与 STM32 联调版本）：**

```
TIME:14:30:05,IP:192.168.1.100
```

**含闹钟配置：**

```
TIME:14:30:05,IP:192.168.1.100,A1:08:00:1,A2:--:--:0,A3:15:45:0
```

**闹钟触发：**

```
TIME:08:00:01,IP:192.168.1.100,EVENT:RING1
```

## STM32 上行（超时未取药）

```
STM32:CMD:REMIND_TAKE_MED,SLOT:1
```

详见 [`esp8266_remind_protocol.md`](esp8266_remind_protocol.md)。

## 注意事项

- 仅 **2.4GHz Wi-Fi**
- 波特率 **9600**，与 STM32 一致
- 供电留足 3.3V 电流余量（约 300–400mA 峰值）
